Proposal:

Deployment

  • After signup and payment workflow at Development for ADT assigning.
  • MW shows customer nearest all ADT.
  • Need to add option add ADT port in MW which is mandatory to run closures.

Outdoor Relocation

  • After signup and payment workflow at Development for ADT assigning.
  • MW shows customer nearest all ADT.
  • Need to add option add ADT port in MW which is mandatory to run closures.

Migration

  • A person who requires migration must be provided a new ADT with a port.
  • In migration workflow nearest all ADT will be shown.
  • Any update and changing of ADT and port rights open to the closure person.

Termination

  • Termination workflow closes once the device is submitted in the store and acknowledgement of the store person.
  • A ADT port free workflow creates after closure of termination workflow.
  • NMS port release after the port free workflow closure.
